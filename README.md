# CULLNO

大量の TGA / PNG 画像から素早くベストショットを選別するデスクトップツール。

## 特徴

- **連射検出** — 連番ファイルを自動グルーピング、展開して比較
- **高速サムネイル** — Sharp + Worker によるバックグラウンド生成・キャッシュ
- **TGA 対応** — 自前デコーダで TGA を直接読み込み
- **ピック & 削除** — お気に入りをマーク、不要な画像はゴミ箱へ即削除
- **エクスポート** — ピック済み画像を任意フォルダにコピー
- **比較ビュー** — 2枚並べて拡大・パン同期で細部を比較
- **キーバインドカスタマイズ** — ほぼ全操作のショートカットを変更可能
- **テーマ切替** — ダーク / ライトモード
- **UI スケール** — 80% 〜 150% で調整可能
- **ホーム背景画像** — お好みの画像を Welcome 画面の背景に設定

## 技術スタック

- Electron 33 + Vite 5
- React 18 + Fluent UI v9
- Zustand 5（状態管理）
- Sharp（サムネイル生成）

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

`release/` に zip が出力されます。
